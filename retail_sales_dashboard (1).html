CREATE TABLE transactions (
  customer_id   VARCHAR(10) NOT NULL,
  trans_date    DATE NOT NULL,
  response      TINYINT(1) DEFAULT 0,
  month         INT,
  month_year    VARCHAR(7),

-- Load from CSV

INTO TABLE transactions
FIELDS TERMINATED BY ','
IGNORE 1 ROWS;

# Phase 2: Data Cleaning with pandas
import pandas as pd

main = pd.read_csv('MainData.csv')
add = pd.read_csv('AddAnlys.csv')

# 1. Check for nulls
print(main.isnull().sum()) # All zeros — clean!

# 2. Parse dates properly
main['trans_date'] = pd.to_datetime(main['trans_date'], dayfirst=True)
add['recency'] = pd.to_datetime(add['recency'], dayfirst=True)

# 3. Outlier check using IQR
Q1 = main['tran_amount'].quantile(0.25)
Q3 = main['tran_amount'].quantile(0.75)
IQR = Q3 - Q1 # = 36
outliers = main[(main['tran_amount'] < Q1 - 1.5*IQR) | (main['tran_amount'] > Q3 + 1.5*IQR)]
# Result: 0 outliers beyond IQR bounds

# 4. Add calculated fields
main['year'] = main['trans_date'].dt.year
main['quarter'] = main['trans_date'].dt.quarter
main['day_of_week'] = main['trans_date'].dt.day_name()
Dataset Quality: Excellent
Zero nulls, zero duplicates across both CSVs. Transaction amounts bounded between ₹10–₹105 with no extreme outliers by IQR method.
February Dip Pattern
February consistently shows ~10–12% lower transaction volume vs other months, likely due to fewer days. Normalised per-day figures confirm no seasonal issue.
Partial Years
2011 starts May, 2015 ends March. Annual comparisons exclude these partial years; monthly analysis uses full 2012–2014 cohort.

🔍 Revenue Stability
Full-year revenue is remarkably stable: ₹2.11M (2012), ₹2.14M (2013), ₹2.09M (2014). No seasonal spikes — consistent demand throughout the year.
🔍 P0 Segment Dominates
P0 (Premium) customers represent 64% of the base but generate ~80% of total revenue (₹6.5M). Their avg. transaction is ₹70 vs ₹50 for P2.
🔍 High Purchase Frequency
70% of customers buy 11–25 times. Average frequency is 18.2 purchases, indicating a loyal, habitual customer base with low churn risk.
🔍 Improving Response Rate
Campaign response grew from 9.8% (2011) to 12.6% (2014) — a 28% improvement. Marketing effectiveness is trending upward year-on-year.
🔍 Cohort Concentration
94.4% of customers made their first purchase in 2011. Acquisition has slowed significantly — only 12 new customers started in 2013.
🔍 All Customers Repeat
100% of 6,884 customers are repeat buyers. Zero one-time purchasers — outstanding retention for a retail chain.

📌 Key Finding #1
Revenue is highly stable at ~₹2.1M/year (2012–2014) — a mature, loyal base. Growth will come from new customer acquisition, not spend-per-customer increases.
📌 Key Finding #2
P0 customers are disproportionately valuable. With 80% of revenue from 64% of customers, a small P2→P0 conversion program could significantly lift total revenue.
📌 Key Finding #3
Campaign response is improving (+28% over 4 years) but still only 11.1% overall. Personalisation and targeting by segment could realistically push this to 15–18%.
